Welcome to the MBI Snark Discord server documentation!
===================================

**MBIS** is a Discord server dedicated to casual educational commentary regarding the Munchausen by Internet phenomenon.
Before you can view or participate in any of the discussion channels, you will need to read and agree to the rules listed in the 
``#server-rules`` channel, located in the ``WELCOME!`` folder on the lefthand navigation bar.

Check out the :doc:`usage` section for further information, including
how to change your personal :ref:`notifications` settings for various :ref:`threads`, or
help grow our community :ref:`chree`!

.. note::

   This project is under active development.
